# 딥다이브 · DeepDive

공부 세션을 **바다 속 잠수**로 기록하는 학습 트래커 웹앱입니다.
타이머를 켜면 집중한 시간만큼(집중 1분 = 수심 1m) 더 깊이 내려가고,
도달한 수심대에서 심해 생물이 해금됩니다. 쌓인 잠수 기록은 학습 분석 대시보드가 됩니다.

## 기능

- **다이브** — 과목·목표 수심을 정하고 잠수(집중) 시작. 일시정지/재개, 탭 전환·새로고침에도 시간 유지
- **항해 로그** — 기간·과목·수심 필터에 반응하는 차트(도넛·라인·히트맵·게이지) + KPI(총 집중시간·최고 수심·연속 잠수일)
- **심해 도감** — 수심대별로 해금되는 생물 컬렉션
- **설정 / 데이터** — 더미 데이터 채우기·초기화, JSON 백업·복원
- **테스트용 주입** — `pnpm dive <수심>` 으로 원하는 수심의 잠수를 바로 기록

## 실행

```bash
pnpm install
pnpm run dev      # http://localhost:3000
```

## 스택

Next.js 16 (App Router) · React 19 · TypeScript · Tailwind CSS v4 · SQLite (node:sqlite)
